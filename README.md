# Agentic-AI-Research-Code 

# Agentic AI Workflow for Software Vulnerability Discovery and Remediation

## Overview

This project aims to develop an Agentic AI system capable of identifying, explaining, reproducing, and remediating software vulnerabilities. The goal is to investigate how effectively an autonomous AI workflow can perform vulnerability management tasks compared to existing Agentic AI systems.

## Current Plan

The project is currently in the research and planning phase.

Development will focus on:

* Selecting open-source models from Hugging Face
* Connecting to models through APIs
* Using Python libraries to build the agent workflow
* Developing agents for vulnerability analysis and remediation
* Evaluating performance using vulnerability benchmarks

## Technologies

Planned technologies include:

* Python
* Hugging Face APIs
* Transformers
* LangChain/LangGraph
* Other supporting machine learning and cybersecurity libraries

## Goals

The final system should be able to:

1. Detect software vulnerabilities
2. Explain identified vulnerabilities
3. Reproduce vulnerabilities when possible
4. Suggest potential fixes
5. Compare results against existing Agentic AI approaches

## Status

Work in progress. The architecture, models, and implementation details may change as research and development continue.
